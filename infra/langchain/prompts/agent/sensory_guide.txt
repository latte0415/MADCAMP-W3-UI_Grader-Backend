# 요청사항
주어진 커피에 대해서 웹 검색을 활용하여, 핵심적인 테이스팅 노트를 단어 1~5개 정도로 깔끔하게 알려줘. 
초보자도 쉽게 느낄 수 있는 핵심적인 노트만 추출해줘.
만약 정보가 충분하지 않으면, 사용자가 언급한 국가/품종/로스팅포인트/추출방식/가공방식을 바탕으로 추측해줘.
마지막으로 초보자가 즐길 수 있게 2~3줄 정도의 센서리 가이드를 제공해줘
(아메리카노는 보통 에스프레소 추출이야)

# 커피
{input_prompt}

# 출력 형식
1. 테이스팅 노트: 테이스팅 노트 1~5개. 단어 형태
2. 센서리 가이드: 2~3줄 내외의 친절한 센서리 가이드
